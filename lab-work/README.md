# Modern CSS Layout Lab

## Objective
This lab demonstrates modern CSS layout techniques including:
- Positioning
- Flexbox
- Grid
- Media Queries

## Modules Covered
1. CSS Positioning (fixed, relative, absolute, sticky)
2. Flexbox for navigation and alignment
3. CSS Grid for dashboard layout
4. Responsive design using media queries

## Features
- Fixed header and footer
- Notification badge using absolute positioning
- Flexbox navigation bar
- Grid-based dashboard
- Fully responsive layout

## Technologies
- HTML5
- CSS3

## Learning Outcome
Students can confidently choose the correct layout technique for real-world UI problems.
